# suomiahven
An extendable Finnish Discord bot. 

## Currently implemented extensions (cogs)
    - ping (in-progress)
    - greeter (in-progress)
    - music (in-progress)
    - help (in-progress)
    - template (reference)

## Dependencies
* Core
    - discord.py
    - python-dotenv
* Music cog
    - yt-dlp
    - pyttsx3
    - pydub
    - PyNaCl

## Running the bot:

Run the `run.py` installer or install the files via `pip3 install -r requirements.txt` and execute the `main.py` directly.
